# Pangantara — BCC Intern Project (1 Bulan)

Aplikasi web yang dikembangkan dalam program **BCC Intern selama 1 bulan**.  
Pangantara menjadi jembatan kolaborasi antara **SPPG** dan **Mitra**, dengan **Admin** sebagai pihak yang membantu pengelolaan, validasi, serta kelancaran proses kerja sama.

---

## Ringkasan Peran (Roles)

Website ini memiliki 3 jenis pengguna:

- **User SPPG**
  - Mengelola kebutuhan/permintaan kerja sama
  - Berkoordinasi dan memantau proses yang berjalan bersama Mitra

- **Mitra**
  - Menangani permintaan/kerja sama dari SPPG
  - Memberi respons, pembaruan status, dan progres sesuai alur kerja

- **Admin**
  - Membantu operasional kerja sama SPPG ↔ Mitra
  - Mengelola data penting, memastikan proses berjalan rapi dan terkontrol

---

## Tujuan Aplikasi

Dengan adanya Pangantara, proses kerja sama menjadi:

- Lebih **terstruktur** (alur jelas, peran jelas)
- Lebih **transparan** (status/progres mudah dipantau)
- Lebih **efisien** (mengurangi komunikasi yang tercecer)
- Lebih **mudah dikelola** (dibantu admin sebagai penghubung dan pengawas proses)

## Cara Membuka Project di IDE

### 1) Clone repository
```bash
git clone https://github.com/fahrypratama17/pangantara.git
cd pangantara
```

### 2) Buka di IDE
**VS Code**
```bash
code .
```

**WebStorm / IntelliJ**
- `File` → `Open...` → pilih folder `pangantara`

### 3) Install dependencies
Gunakan salah satu (sesuaikan project):

```bash
npm install
# atau
yarn
# atau
pnpm install
```

### 4) Jalankan aplikasi (development)
Biasanya salah satu dari ini:

```bash
npm run dev
# atau
npm start
```
